## 1. The Problem

The current sorry has this statement:

```lean
private theorem _root_.ValuationSpectrum.tate_vle_one_on_A₀_isContinuous_sorry
    {A : Type*} [CommRing A] [TopologicalSpace A] [PlusSubring A] [IsHuberRing A]
    [IsTateRing A] [IsNoetherianRing A] [IsDomain A] (D' : RationalLocData A) :
    ∀ (w : ValuativeRel A), (∀ a ∈ D'.P.A₀, w.vle a 1) →
      (⟨w⟩ : Spv A).IsContinuous
```

**This statement is FALSE as written.** The trivial valuation `w` (with `w(a) = 1`
for all `a ≠ 0`) satisfies `∀ a ∈ D'.P.A₀, w.vle a 1` vacuously, but is only
continuous on `A` iff `A` is discrete. For non-discrete Tate rings this fails.

The correct characterization is Wedhorn Theorem 7.10:

> **Theorem 7.10 (Wedhorn).** Let `A` be an f-adic ring with pair of definition
> `(A₀, I)`. A valuation `v : Spv A` is continuous iff
> 1. `v(a) ≤ 1` for all `a ∈ A₀`, AND
> 2. `v(a) < 1` for all `a ∈ I`.

Condition (2) is what the current sorry is missing. For trivial `v`, condition (2)
fails on any non-zero ideal of definition, so the trivial `v` is correctly
classified as non-continuous.

```lean
theorem Valuation.isContinuous_of_le_one_and_pow_cofinal
```

This is the **backward direction** of Wedhorn 7.10 (given all three conditions,
conclude continuity). It takes a classical `Valuation A Γ₀` (not `ValuativeRel`).

```lean
    (comap φ v).IsContinuous
```

This is the **functoriality of continuity**: continuous pullback along a
continuous ring hom preserves continuity of `Spv` points.

The `algebraMap` from `A` to `Localization.Away D.s` is continuous wrt the

## 5. Optional: Actually Proving Wedhorn 7.18

The sorry in Step 1 is Wedhorn 7.18. Its proof requires:

### 5.1 Key lemma (Wedhorn 7.22)

> **Lemma 7.22.** Let `A` be an f-adic ring with pair of definition `(A₀, I)`.
> Let `B` be an open subring of `A` with `A₀ ⊆ B`. Let `V` be a valuation
> subring of `Frac(A)` with `B ⊆ V` (image-wise). Then the comap valuation
> `V.valuation.comap (algebraMap A Frac(A))` is continuous on `A`.

**Proof sketch:**
1. `A₀ ⊆ B ⊆ V`, so the comap `w` satisfies `w ≤ 1` on `A₀`.
2. Since `B` is open and bounded (image of bounded subring under continuous map),
   `w` additionally satisfies a cofinal condition on `I`.
3. Specifically: for any topologically nilpotent `π ∈ I`, `π^n → 0` in `A`, so
   `w(π^n) → 0` in the value group of `V.valuation`.
4. Take `g = w(π)`. Then `g^n` is cofinal.
5. Apply `Valuation.isContinuous_of_le_one_and_pow_cofinal`.

The tricky step is #3: we need `w(π) < 1` strictly. This follows from:
- `π^n → 0` in `A` (topological nilpotence)
- If `w(π) = 1`, then `w(π^n) = 1` for all `n`, contradicting `w(π^n) → 0`
  (since `w` as a classical valuation has `w⁻¹({ < 1 })` containing a nhd of 0
  by CONTINUITY, which we're trying to prove — circular!).

**Resolution of circularity:** `w(π^n) → 0` in the **sense of the value group**
is a separate condition from topological continuity. In Wedhorn's framework,
this is proved by direct computation: for valuation subrings `V`, the value
group is totally ordered, so `w(π) ≤ 1` and `π` non-unit in `V` forces
`w(π) < 1`.

Is `π` non-unit in `V`? Yes! `π` is a topologically nilpotent unit in `A`, but
in `V` (a valuation subring of `Frac(A)`), `π` is topologically nilpotent —
and in a valuation subring, topologically nilpotent elements are in the
maximal ideal.

Wait — this is circular too. "Topologically nilpotent in `V`" requires a topology
on `V`, which we don't have a priori.

**Correct resolution:** Use the fact that `π` is a unit in `A` but `π ∈ I` (ideal
of definition). For any valuation subring `V ⊇ A₀`, the image of `I` in `V` is
contained in the maximal ideal of `V` (because `I` is topologically nilpotent,
hence `π` and any `V`-unit `v` with `v · π^n → 0` gives `v = 0`, contradiction).

More precisely: since `π` is topologically nilpotent in `A` and `V` extends `A₀`'s
valuation, `V.valuation(π) < 1`. This requires `V` to "know about" the topology of
`A`, which is precisely the content of Wedhorn 7.22.

### 5.2 Alternative: Construct the valuation directly

Instead of pulling back from `Frac(R)`, construct `V` directly on `R = Localization.Away D'.s`
by localizing at a prime containing `x` (the test element) but not containing
`s` (the localization element). This gives a valuation on `R` with the right
properties, and continuity can be verified directly from the construction.

This approach is used in the discrete case (see `exists_mem_spa_supp_eq_of_prime`
in `AdicSpectrum.lean`). For the non-discrete case, one uses Wedhorn Lemma 7.45
(proved sorry-free in `Lemma745.lean`).

## 7. Risk Analysis

### 7.1 Circularity risk
The existing `locLift_vle_one_at_spa` uses `hv_cont` (the false sorry) in a way
that passes through rational-open membership. The refactored version uses
`comap_isContinuous`, which requires `v` continuous on `Loc`. We need to verify
that the user of `locLift_vle_one_at_spa_cont` (the Tate instance) can actually
supply this continuity.

**Mitigation:** The topology-aware `isIntegral_of_forall_continuous_valuation_le_one`
only invokes `hvle` for continuous `v`. So in the Tate instance, when we apply
`locLift_vle_one_at_spa_cont`, we're in a context where `v` is guaranteed continuous
by the outer `hvle` hypothesis. No circularity.

### 7.2 Instance binding risk
`CompatiblePlusSubring` is a new typeclass requirement. Adding it to
`locLift_vle_one_at_spa_cont` means users of this new version need to provide it
at the call site.

**Mitigation:** Only `HasLocLiftPowerBounded.tate` calls `locLift_vle_one_at_spa_cont`,
and that instance already takes `[CompatiblePlusSubring A]`. No broader propagation
needed.

### 7.3 Wedhorn 7.18 proof difficulty (M6)
The adic Nullstellensatz proof requires non-trivial mathematical machinery.
Attempting M6 could exceed the effort budget.

**Mitigation:** M1-M5 deliver value independent of M6. The sorry becomes a single
clearly-labeled "Wedhorn 7.18" sorry, which is a well-known target. M6 can be
a separate work item.
