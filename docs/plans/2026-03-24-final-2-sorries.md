## Sorry 1: p-torsion-freeness of A° (line 346)

### Statement
```lean
hp_reg : ∀ (a : O), (p : O) * a = 0 → a = 0
```
where `O = ↥(powerBoundedSubring.toSubring A)` and A is a perfectoid ring.

### Mathematical Status

**This claim may be FALSE for general perfectoid rings.** For perfectoid
fields, A° ⊆ K (a field) is a domain, so p is a non-zerodivisor. But
for general perfectoid rings, A° could have p-torsion.

### Two approaches

**Approach A: Prove it for the cases where it holds.**

For a perfectoid ring A with `p = c · ϖ^p` (ϖ a unit in A):
- `p * a = 0` in A° means `c * ϖ^p * (a : A) = 0` in A.
- Since ϖ is a unit: `c * (a : A) = 0`.
- If c is a non-zerodivisor in A: `(a : A) = 0`, hence `a = 0` in A°.
- c is a non-zerodivisor when A is a domain, or more generally when the
  natural map A° → A is injective (always true) and c ∉ ann(A).

**The question: is c a non-zerodivisor in A for perfectoid rings?**

If c were a zerodivisor: ∃ nonzero b ∈ A with c · b = 0. Then
p · b = c · ϖ^p · b = ϖ^p · (c · b) = 0. So b is p-torsion.
And ϖ^p · b = (1/c) · p · b = 0... but 1/c doesn't exist.

For the Hausdorff argument: if p · a = 0 in A°, does a = 0?
In A: p · (a : A) = 0. The set ann(p) in A is the set of elements
killed by p. For a T₀ topological ring where p is topologically
nilpotent: ann(p) ⊆ ∩_n p^n · A = {0} by Hausdorffness... but
that's wrong: ann(p) is NOT contained in ∩ p^n · A in general.

**Approach B: Restructure the proof to avoid hp_reg.**

The Berkeley Lectures proof does NOT use p-regularity. It uses the
quotient computation: `W(k)/(ξ, [ϖ]) = k/ϖ = A°/ϖ♯`. The key
step that uses p-regularity in our proof (line 422: showing r ∈ ker(θ)
from p · θ(r) = 0) can be replaced by the quotient approach.

**Specific restructuring:**
1. Replace the `cancel_pow` + `hp_reg` argument (lines 369-380) with:
   for x ∈ ker(θ) with x.coeff 0 = 0: x ∈ (p) in W(k). Write x = p·y.
   Then θ(p·y) = p·θ(y) = 0. Instead of concluding θ(y) = 0, use:
   y ≡ some element of ker(θ) (mod p) by the division step.
2. This avoids needing p-regularity in A° entirely.
3. The trade-off: the proof becomes more circular (the Nakayama argument
   and the division step become interdependent).

**Approach C: Add as a hypothesis to IsPerfectoidRing.**

Add `p_regular : ∀ (a : A), (p : A) * a = 0 → a = 0` to the class.
This holds for all "standard" perfectoid rings (fields, domains, etc.)
and simplifies the proof. The trade-off: strengthens the class.

### Recommended approach: B (restructure)

This matches the Berkeley Lectures proof most closely and works for
all perfectoid rings without additional hypotheses.

### Estimated effort: ~30 lines to restructure

## Sorry 2: Divisibility in the tilt (line 429)

### Statement
```lean
∃ q₀, y.coeff 0 = ξ.coeff 0 * q₀
```
where `ξ, y ∈ ker(θ)` with `ξ.coeff 0 ≠ 0`, in `k = tilt p A`.

### Mathematical Content

Both ξ.coeff 0 and y.coeff 0 are in the kernel of
`Perfection.coeff 0 : Perfection(A°/(p)) → A°/(p)` (since their
untilts are in (p), hence their 0-th Perfection components are 0).

The kernel of `Perfection.coeff 0` is:
```
ker(coeff_0) = {(0, a₁, a₂, ...) : a_{n+1}^p = aₙ, a₁^p = 0}
```

In a perfect ring of char p, this kernel is the image of the
INVERSE Frobenius applied to the kernel of the identity (i.e.,
the "shifted" elements). More precisely:
```
ker(coeff_0) = F⁻¹(Perfection(R)) · ϖ̃
```
where ϖ̃ = (0, ϖ̄, ϖ̄^{1/p}, ...) is the canonical element
(the "tilt of the pseudo-uniformizer").

The divisibility ξ.coeff 0 | y.coeff 0 in ker(coeff_0) follows from:
- ξ.coeff 0 and y.coeff 0 are both in ker(coeff_0)
- ker(coeff_0) is a principal ideal generated by ϖ̃ (or any
  element whose 1st component generates the nilradical of A°/(p))
- Since ξ.coeff 0 ≠ 0 and lies in this principal ideal, it's a
  unit multiple of the generator times some power of the Frobenius

### Key Mathlib APIs needed

1. `Perfection.coeff 0` : `Perfection R p → R` (ring hom at index 0)
2. The kernel of `coeff 0` on Perfection(R) — need to characterize
3. The Frobenius acts on this kernel — need shift lemma
4. Principality of ker(coeff_0) — the core claim

### Approach

1. Show both ξ.coeff 0 and y.coeff 0 have Perfection.coeff 0 = 0.
   (From: x ∈ ker(θ) → θ([x.coeff 0]) = (x.coeff 0).untilt ∈ (p)
   → mk(untilt) = Perfection.coeff 0 = 0.)

2. Characterize ker(Perfection.coeff 0) on Perfection(R):
   ```
   a ∈ ker(coeff_0) ↔ a = F⁻¹(b) for some b with b_0 nilpotent
   ```
   where F⁻¹ is the inverse Frobenius (shift forward).

3. In the tilt of a perfectoid ring: ker(coeff_0) is generated by
   a single element (the tilt of the pseudo-uniformizer).
   This uses: the nilradical of A°/(p) is generated by ϖ̄ (image
   of the pseudo-uniformizer), and in the perfection, the "lifted"
   nilradical is principal.

4. Since ker(coeff_0) is principal and ξ.coeff 0 ≠ 0 is in it,
   ξ.coeff 0 = (generator) * (unit). Then y.coeff 0 = (generator) * q,
   so y.coeff 0 = ξ.coeff 0 * (unit⁻¹ * q).

### Estimated effort: ~50 lines

The characterization of ker(Perfection.coeff 0) requires understanding
the perfection's structure. The principality of this kernel for
perfectoid rings uses the pseudo-uniformizer.
