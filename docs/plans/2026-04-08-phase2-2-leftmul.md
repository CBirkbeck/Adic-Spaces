## The statement to prove

```lean
theorem tateAlgNhd_leftMul [IsTateRing A] (P : PairOfDefinition A)
    (x : ↥(TateAlgebra A)) (i : ℕ) :
    ∃ j, (tateAlgNhd P j : Set ↥(TateAlgebra A)) ⊆
      (x * ·) ⁻¹' (tateAlgNhd P i : Set ↥(TateAlgebra A))
```

Where:
- `tateAlgNhd P n` = image of `(pairIdeal P)^n` in `TateAlgebra A` under the
  subring inclusion.
- `pairIdeal P` = `Ideal.map (pairConstantHom P) P.I`, the ideal of `pairSubring P`
  generated by constant series from `P.I`.

---

## Route B: coefficient bookkeeping (recommended)

We avoid constructing the localization iso `A⟨X⟩ ≃ A₀⟨X⟩[1/π]` and instead
prove the inclusion directly via coefficient computations.

**High-level strategy.** For `x = Σ a_l X^l ∈ TateAlgebra A`:
1. Since `x` is a restricted power series, `a_l → 0` in `A`, so only finitely
   many `a_l` are not already in `P.I^i`.
2. For each "bad" `a_l` (with `a_l ∉ P.I^i`), continuity of mul-by-`a_l` in A's
   topology gives `m_l` with `a_l · P.I^{m_l} ⊆ P.I^i`.
3. Take `j := max(i, sup {m_l : l bad})`.
4. For `y ∈ tateAlgNhd P j`, show `x · y ∈ tateAlgNhd P i` by checking
   coefficient-by-coefficient.

### Sub-task A: Coefficient extraction for `tateAlgNhd P n` (forward direction)

**Estimated:** ~40 lines.
**Dependencies:** none — pure algebra on the existing `tateAlgNhd` definition.
**Location:** `Adic spaces/TateAlgebraTopology.lean` (new helper lemmas near
`tateAlgNhd`).

**Lemma statement:**
```lean
theorem tateAlgNhd_coeff_mem (P : PairOfDefinition A) (n : ℕ)
    {y : ↥(TateAlgebra A)} (hy : y ∈ tateAlgNhd P n) (l : ℕ) :
    MvPowerSeries.coeff (Finsupp.single 0 l) y.val ∈
      ((P.I ^ n : Ideal P.A₀) : Set P.A₀).image Subtype.val
```

(Alternatively: `∃ b ∈ P.I^n, (Subtype.val b : A) = coeff l y.val`.)

**Proof:**
1. Unpack `hy : y ∈ tateAlgNhd P n`. By definition, `y = ↑z` for some
   `z ∈ (pairIdeal P)^n`.
2. Use `Submodule.span_induction` on `z ∈ (pairIdeal P)^n`:
   - **Base case:** `z` is an `n`-fold product of generators `a_1 · ... · a_n · g`
     where `a_k ∈ pairConstantHom P.I` (constants in `pairSubring P`) and
     `g ∈ pairSubring P`. Compute the `l`-th coefficient:
     `coeff l (a_1 · ... · a_n · g) = (a_1 · ... · a_n : A) · (coeff l g : A₀)`.
     Since `a_1 · ... · a_n ∈ P.I^n ⊆ A₀`, the product is in `P.I^n`.
   - **Zero case:** trivial.
   - **Additive case:** sum of coefficients in `P.I^n` stays in `P.I^n`.
   - **Scalar mul case:** multiplication by `r ∈ pairSubring P` scales the
     coefficient; the inner ideal `P.I^n` is closed under left multiplication
     by `A₀` (which includes `coeff l r`).
3. Extract the witness `b ∈ P.I^n`.

**Pitfalls:** The subtype coercions are fiddly. Use `MvPowerSeries.coeff_mul`
for the base-case product expansion.

### Sub-task B: I-adic continuity for `A`

**Estimated:** ~20 lines.
**Dependencies:** none — uses existing `IsTopologicalRing` + `PairOfDefinition`.
**Location:** `Adic spaces/HuberRings.lean` (if not already there) or a new
helper in `TateAlgebraTopology.lean`.

**Lemma statement:**
```lean
theorem PairOfDefinition.exists_mul_mem_pow_of_mul
    (P : PairOfDefinition A) (a : A) (n : ℕ) :
    ∃ m : ℕ, ∀ b : P.A₀, b ∈ P.I ^ m →
      ∃ c : P.A₀, c ∈ P.I ^ n ∧ (a * (b : A) : A) = (c : A)
```

**Proof:**
1. Continuity of mul-by-`a` at 0 in A: `(a * ·) ⁻¹' U ∈ nhds 0` for any `U ∈ nhds 0`.
2. Take `U` = image of `P.I^n` in A (a nhd of 0 by `P.hasBasis_nhds_zero`).
3. The preimage contains some image of `P.I^m` (by the same basis).
4. Extract `m` and produce the witness `c = a * b` (which is in `P.I^n · A₀`
   when translated back).

**Alternative simpler statement (may be enough):**
```lean
theorem exists_mul_subset_pow (P : PairOfDefinition A) (a : A) (n : ℕ) :
    ∃ m : ℕ, (a • ((P.I ^ m : Ideal P.A₀) : Set P.A₀).image Subtype.val) ⊆
      ((P.I ^ n : Ideal P.A₀) : Set P.A₀).image Subtype.val
```

**Pitfalls:** The set-level vs ideal-level bookkeeping is the main source of
friction. Use `P.hasBasis_nhds_zero` directly.

### Sub-task C: Almost-all-coeffs-in-`I^n` for restricted power series

**Estimated:** ~15 lines.
**Dependencies:** none — uses `TateAlgebra.IsRestricted` (the defining property
of restricted power series).
**Location:** `Adic spaces/TateAlgebra.lean` or a new helper in
`TateAlgebraTopology.lean`.

**Lemma statement:**
```lean
theorem TateAlgebra.coeff_eventually_mem_pow
    (P : PairOfDefinition A) (x : ↥(TateAlgebra A)) (n : ℕ) :
    ∃ N : ℕ, ∀ l ≥ N, ∃ b : P.A₀, b ∈ P.I ^ n ∧
      (b : A) = MvPowerSeries.coeff (Finsupp.single 0 l) x.val
```

**Proof:**
1. `x` is restricted, so `coeff_l x.val → 0` in `A` as `l → ∞`. Unpack
   `TateAlgebra.IsRestricted` to get this.
2. `P.I^n` (image in A) is a nhd of 0 by `P.hasBasis_nhds_zero`.
3. Eventually (for `l ≥ N`), `coeff l x.val ∈ P.I^n` (as a nhd of 0 fact).
4. Extract the witness `b ∈ P.I^n`.

**Pitfalls:** `IsRestricted` for `MvPowerSeries` might be stated in terms of
convergence in the product topology on MvPowerSeries, which needs to be related
to A's topology. Check `MvPowerSeries.IsRestricted` carefully.

### Sub-task D: Reverse coefficient characterization (principal case)

**Estimated:** ~50 lines.
**Dependencies:** Sub-task A (for the forward direction identity).
**Location:** new section in `Adic spaces/TateAlgebraTopology.lean`.

**Lemma statement (principal case):**
```lean
theorem tateAlgNhd_mem_of_coeff_mem_principal
    (P : PairOfDefinition A) (π : A) (hπ_mem : π ∈ P.A₀)
    (hπ_gen : P.I = Ideal.span {⟨π, hπ_mem⟩}) (n : ℕ)
    (y : ↥(TateAlgebra A)) (hy_pair : y ∈ pairSubring P)
    (hy_coeff : ∀ l, ∃ b : P.A₀, b ∈ P.I ^ n ∧
      (b : A) = MvPowerSeries.coeff (Finsupp.single 0 l) y.val) :
    y ∈ tateAlgNhd P n
```

**Proof (principal case):**
1. By `hy_coeff`, each `coeff l y.val = π^n · c_l` for some `c_l ∈ A₀`.
2. Define `g : A₀⟨X⟩` (= `pairSubring P`) with `coeff l g = c_l`. Need to check
   convergence: `c_l → 0` in the `P.I`-adic topology on `P.A₀`.
3. Convergence: from `y` being restricted, `coeff l y.val → 0` in A. So
   `π^n · c_l → 0` in A. Multiplication by `π^n` is continuous (A is a topological
   ring), and in fact a homeomorphism onto its image for nonzero `π` (via the
   inverse `π^{-1}` available in a Tate ring). Hence `c_l → 0` in A, and a fortiori
   in `A₀` with the subspace topology.
4. Define `g ∈ pairSubring P` with these coefficients (needs the pairSubring
   constructor for restricted series from a convergent coefficient sequence).
5. Then `y.val = π^n · g.val` in MvPowerSeries (coefficient-wise).
6. So `y = (π^n) · g` in `pairSubring P`, where `π^n ∈ P.I^n` as a constant series.
7. Hence `y ∈ (pairIdeal P)^n` (principal ideal version of the ideal power).

**Pitfalls:**
- Lean's `Ideal.span` vs `Ideal.map` bookkeeping.
- The convergence argument (step 3) relies on the Tate ring structure to get
  `π` invertible in A. In a general Huber ring this wouldn't hold.
- Step 4 (constructing `g` from a convergent coefficient sequence) needs
  `pairSubring P` constructor; check what's available.

### Sub-task E: Pair-of-definition refinement (Wedhorn 6.14)

**Estimated:** ~30 lines.
**Dependencies:** none — pure Wedhorn 6.14 for our specific setup.
**Location:** `Adic spaces/HuberRings.lean`.

**Lemma statement:**
```lean
theorem IsTateRing.exists_principal_pairOfDefinition
    (A : Type*) [CommRing A] [TopologicalSpace A] [IsTateRing A] :
    ∃ (P : PairOfDefinition A) (π : A) (hπ_mem : π ∈ P.A₀),
      P.I = Ideal.span {⟨π, hπ_mem⟩} ∧
      IsTopologicallyNilpotent π ∧
      IsUnit π
```

**Proof (via Wedhorn 6.14):**
1. `IsTateRing A` gives us `∃ u : Aˣ, IsTopologicallyNilpotent (u : A)`.
2. `IsHuberRing A` gives us some pair `P' : PairOfDefinition A`.
3. Construct a NEW pair `P` with `P.A₀` containing `u`, and `P.I` generated by `u`.
   - `P.A₀ := P'.A₀[u]` (sub-algebra generated) — need `u` bounded (Wedhorn 6.4).
   - `P.I := (u : P.A₀) · P.A₀`.
   - Verify `P` is a pair of definition: open, f.g., adic.
4. Package as a witness.

**Pitfalls:** This is a non-trivial lemma but it's standard. The main challenge
is showing `P.A₀` is open (follows from `P'.A₀` being open and `P.A₀ ⊇ P'.A₀`)
and that `P.I`-adic topology matches the existing one (follows from cofinality
of ideal-of-definition powers, Wedhorn 6.4).

**Alternative (shortcut):** Just ADD the principal-pair hypothesis to
`tateAlgNhd_leftMul`. This makes the theorem conditional on having a principal
pair, which is `∀ pair, ∃ principal pair with same topology`. We'd need a
change-of-pair lemma for the topology, but that's also non-trivial.

**Recommendation:** Do Sub-task E properly — it's reusable elsewhere in the
project and Wedhorn 6.14 is a named result.

### Sub-task F: Assembly of the hard case

**Estimated:** ~50 lines.
**Dependencies:** Sub-tasks A, B, C, D, E.
**Location:** `Adic spaces/TateAlgebraTopology.lean` (fill the sorry).

**Proof:**
1. By Sub-task E, choose a principal pair `P_p = (P_p.A₀, P_p.I = (π))` with
   the same topology as `P`. **Wait** — we need the topology on `TateAlgebra A`
   defined via `P` to match that via `P_p`. This is an additional topology
   coincidence lemma (easy, both are Tate topologies).
2. Alternatively: prove the leftMul for `P_p` first (using principal structure
   from Sub-task D), then transport to `P`.
3. For `P_p`, apply Sub-tasks B, C:
   - Let `x ∈ TateAlgebra A`.
   - Let `S := {l : coeff l x ∉ P_p.I^i}`. By Sub-task C, `S` is finite.
   - For each `l ∈ S`, by Sub-task B (applied with `a = coeff l x`, `n = i`),
     get `m_l` with `(coeff l x) · P_p.I^{m_l} ⊆ P_p.I^i`.
   - Set `j := max(i, sup {m_l : l ∈ S})`.
4. For `y ∈ tateAlgNhd P_p j`:
   - By Sub-task A, all coefficients of `y` are in `P_p.I^j`.
   - Compute coefficients of `x · y` using `MvPowerSeries.coeff_mul`:
     `coeff_k (x · y) = Σ_{l+m=k} (coeff_l x) · (coeff_m y)`.
   - Each term `(coeff_l x) · (coeff_m y)`:
     - If `l ∉ S`: `(coeff_l x) ∈ P_p.I^i` and `(coeff_m y) ∈ P_p.I^j`, product
       is in `P_p.I^{i+j} ⊆ P_p.I^i`. ✓
     - If `l ∈ S`: `(coeff_m y) ∈ P_p.I^j ⊆ P_p.I^{m_l}`, and by construction
       `(coeff_l x) · P_p.I^{m_l} ⊆ P_p.I^i`. ✓
   - Sum of terms all in `P_p.I^i` is in `P_p.I^i`.
5. Conclude: all coefficients of `x · y` are in `P_p.I^i`, `x · y` is in
   `pairSubring P_p` (since `P_p.I ⊆ A₀`), and `x · y ∈ tateAlgNhd P_p i` by
   Sub-task D.
6. Transport back to the original `P`: the topologies coincide (additional
   lemma), so `tateAlgNhd P_p i ⊆ tateAlgNhd P i` after a finite power shift
   (cofinality of ideals).

**Pitfalls:** Step 6 (transport between pairs) is non-trivial and may
significantly increase the line count.
