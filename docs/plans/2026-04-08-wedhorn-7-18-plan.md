## 1. Wedhorn's Statement

**Proposition 7.18 (Wedhorn).** Let `A` be an f-adic ring. Set
- `ℛ_A := { A' ⊆ A subring : open and integrally closed in A }`
- `𝒮_A := { S ⊆ Cont A : ∃ X ⊆ A, S = S_X = { v ∈ Cont A : v(f) ≤ 1 ∀ f ∈ X } }`

The maps
- `σ : ℛ_A → 𝒮_A`, `A' ↦ { v ∈ Cont A : v(f) ≤ 1 ∀ f ∈ A' }`
- `τ : 𝒮_A → ℛ_A`, `S ↦ { f ∈ A : v(f) ≤ 1 ∀ v ∈ S }`

are mutually inverse bijections.

**Consequence (the form we need):** For an open subring `B ⊆ A` and `f ∈ A`,

> `f ∈ integralClosure(B) ↔ v(f) ≤ 1 for all v ∈ Cont A with v(b) ≤ 1 for all b ∈ B`.

The (⇒) direction is easy. The (⇐) direction is what we need to prove.

## 2. Reusable Infrastructure (already in the codebase)

The proof structure of **Lemma 7.45** (`Adic spaces/Lemma745.lean:344`,
`exists_spa_point_via_restrictToConvex`) is a near-perfect template:

1. **Stacks 090P** for the domination construction (we use `Subring.exists_le_valuationSubring_of_isIntegrallyClosedIn` instead of `exists_valuationSubring_of_prime`).
2. **`g_max` computation** — `Finset.sup'` of `V₀.valuation` over generators of `P.I`. Already used in Lemma 7.45.
3. **`convexGenerated` and `restrictToConvex`** — already proved sorry-free in `ValuationCoarsening.lean` and `ValuationContinuity.lean`.
4. **`withZero_inv_pow_cofinal_of_convexGenerated`** — gives the cofinal property in `WithZero H_gen.toSubgroup`. Already proved sorry-free in `Lemma745.lean:194`.
5. **`Valuation.isContinuous_of_le_one_and_pow_cofinal`** — base continuity criterion. Already in `ValuationContinuity.lean`.
6. **Valuation extension** via `vExtFun_*` lemmas (lines 243–339 of `Lemma745.lean`). Used in Lemma 7.45 for support extension.

**Key difference from Lemma 7.45:**
- Lemma 7.45 input: a non-open prime `𝔭`. Output: Spa point with `𝔭 ≤ supp`.
- **Wedhorn 7.18** input: an element `f̄` not integral over `B`. Output: continuous valuation with `v(f̄) > 1` and `v(B) ≤ 1`.

The two proofs share ~90% of their machinery. The new pieces are the conductor-ideal analysis and the modified domination step.

## 3. Proof Outline for the Existential

We aim to prove (the existential in `Presheaf.lean:1244`):
```
∃ V : ValuationSubring (FractionRing R),
  integralClosure(B) ⊆ V ∧
  ι x ∉ V ∧
  ∃ g < 1, V.valuation ≤ g on image of P.I ∧ g^n cofinal
```

### Step 1: Get a "raw" valuation subring V₀ from Stacks 090P
By `Subring.exists_le_valuationSubring_of_isIntegrallyClosedIn`, since `ι x ∉ integralClosure(B)`,
there exists `V₀ ⊆ Frac(R)` with:
- `integralClosure(B) ⊆ V₀.toSubring`
- `ι x ∉ V₀`

### Step 2: Compute `g_max` on generators of `P.I`
Take a finite set `S` of generators of `P.I` (from `P.fg`). Compute
`g_max := Finset.sup' S (fun t ↦ V₀.valuation (ι (P.A₀.subtype t)))`.

We need `g_max < 1` (strictly), which requires every generator to have `V₀.valuation < 1`.

**Issue:** `V₀.valuation(ι π) ≤ 1` (since π ∈ P.A₀ ⊆ B ⊆ V₀). But strict `< 1` is not automatic — could equal `1` if `ι π` is a `V₀`-unit.

**Solution:** Use a REFINED version of Stacks 090P. Specifically, use `LocalSubring.exists_le_valuationSubring_of_isIntegrallyClosedIn` (Stacks 090P part 2) with a local subring whose maximal ideal contains the image of `P.I`.

### Step 3: Refine V₀ to ensure `g_max < 1`
Construct a local subring `L = LocalSubring.ofPrime (integralClosure B) 𝔭` where `𝔭` is a maximal ideal of `integralClosure(B)` containing both:
- The image of `P.I` (so generators have valuation `< 1` after domination)
- The conductor `S(x) := { s ∈ integralClosure(B) : sx ∈ integralClosure(B) }` (so `ι x ∉ L.toSubring`)

Then apply `LocalSubring.exists_le_valuationSubring_of_isIntegrallyClosedIn` to get
`V` dominating `L` with `ι x ∉ V`. Domination gives image of `P.I ⊆ V.nonunits`.

**Sub-task:** Prove `S(x) + image_in_R₀(P.I)` is a proper ideal of `integralClosure(B)`.
This uses Tate ring properties (topologically nilpotent unit + boundedness).

### Step 4: Prove `g_max < 1` for the refined V
With `V` from Step 3, image of `P.I ⊆ V.nonunits`, so `V.valuation < 1` strictly on every generator.
Hence `g_max < 1`.

### Step 5: Coarsen to `WithZero H_gen.toSubgroup` for cofinality
Following Lemma 7.45's pattern:
1. Set `u_max := Units.mk0 g_max hg_ne0` (provided `g_max ≠ 0`; if `g_max = 0`, all generators are in support, edge case).
2. Set `H_gen := ConvexSubgroup.convexGenerated hu_max_inv_gt1`.
3. Use `restrictToConvex` to coarsen `V.valuation.comap ι` to `WithZero H_gen.toSubgroup`.
4. The coarsened valuation `v_r` has `v_r ≤ 1` on `R` (from `restrictToConvex_le_one`).
5. Cofinal property: by `withZero_inv_pow_cofinal_of_convexGenerated`.

### Step 6: Apply `isContinuous_of_le_one_and_pow_cofinal`
The coarsened valuation `v_r` satisfies all hypotheses of
`Valuation.isContinuous_of_le_one_and_pow_cofinal`:
- `v_r ≤ 1` on `P.A₀` ✓
- `v_r ≤ g_cont` on `P.I` for some `g_cont < 1` ✓
- `g_cont^n` cofinal ✓
Conclude `v_r.IsContinuous`.

### Step 7: Verify `v_r(ι x) > 1`
This is the tricky part: coarsening might collapse `v_r(ι x)` to `1` or `0` if the value of `ι x`
is "outside" the convex subgroup we restricted to.

**Workaround:** Don't coarsen — use the original `V.valuation.comap ι` if it has a MulArchimedean
value group. For non-MulArchimedean, we need a different cofinality argument.

**Alternative:** Re-engineer to track `v_r(ι x) > 1` through the coarsening.

### Step 8: Package the existential
Set the existential's `g := V.valuation((P.A₀.subtype t₀))` for the specific generator `t₀`
realizing `g_max`. Provide the bounds and cofinality from Steps 4–6.

## 6. Risks and Mitigations

### Risk A: W2 (properness of S(x) + image P.I) might require additional hypotheses
- **Mitigation:** Add `[B is bounded in R]` or `[CompleteSpace R]` if needed.
- **Fallback:** Skip W2 and assume the maximal ideal exists as a hypothesis to W3.

### Risk B: W7 (tracking `v_r(x) > 1`) is genuinely subtle
- **Mitigation:** Use a different convex subgroup `H'` that explicitly contains `v(ι x)`.
- **Alternative:** Don't coarsen at all if `V₀` already has MulArchimedean value group.
- **Fallback:** Add `[MulArchimedean V.ValueGroup]` as an existential hypothesis.

### Risk C: Mathlib API gaps
- The `LocalSubring.ofPrime` API may have rough edges. We may need to add bridging lemmas.
- **Mitigation:** Use existing patterns from `LocalSubring.exists_le_valuationSubring_of_isIntegrallyClosedIn` proof.
