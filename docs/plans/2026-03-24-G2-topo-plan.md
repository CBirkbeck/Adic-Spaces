# G2-topo: Wedhorn's T-topology on the Tate Algebra

## Goal

Define Wedhorn's T-topology (Definition 5.48) on the Tate algebra and prove
the universal property (Corollary 5.50). This gives continuous evaluation maps
`A⟨X⟩ → B` when X maps to a power-bounded element, enabling:
- Topological identification `A⟨X⟩/(1-fX) ≃ₜ presheafValue D`
- Strict exactness for Lemma 8.33
- `IsSheafyTopRing` (full topological sheaf condition)

## Background (Wedhorn §5.6)

**Remark 5.47:** On `A[X]`, the T-topology has neighborhoods of 0:
```
U_{[X]} = {∑ aᵥ X^ν : aᵥ ∈ T^ν · U for all ν}
```
For T = {1} (our univariate case): `T^n = {1}`, so `T^n · U = U`.
Hence `U_{[X]} = {∑ aₙ Xⁿ : aₙ ∈ U for all n}`.
This IS the product topology.

**Definition 5.48:** `A⟨X⟩_T` = restricted power series with T-topology.
Neighborhoods: `U_{⟨X⟩} = {∑ aₙ Xⁿ ∈ A⟨X⟩ : aₙ ∈ T^n · U for all n}`.
For T = {1}: `U_{⟨X⟩} = {∑ aₙ Xⁿ : aₙ ∈ U for all n}` = product topology.

**Proposition 5.49(3):** If A complete and T bounded, then A⟨X⟩_T is the
completion of A[X]_T. For T = {1}: A⟨X⟩ = completion of A[X] (product topology).

**Corollary 5.50 (Universal property):** If A complete, T bounded, B complete,
f : A → B continuous, and {f(t)xᵢ} power-bounded in B, then ∃! continuous
g : A⟨X⟩_T → B with g ∘ ι = f and g(Xᵢ) = xᵢ.

## The topology issue

For T = {1}, the T-topology = product topology. With this topology, `X^n → 0`
(since coeff_m(X^n) = δ_{m,n} → 0 for each fixed m). So any continuous map
g : A⟨X⟩ → B sending X ↦ b forces b^n = g(X^n) → 0, i.e., b topologically
nilpotent.

For the R(1/f) case, we need X ↦ 1/f. Is 1/f topologically nilpotent in
presheafValue D? YES if the localization topology makes (1/f)^n → 0. This
happens when 1/f is in the ideal of definition I·D of the localization.

For a Tate ring with pair (A₀, I) and topologically nilpotent unit s:
- The localization A_f has ring of definition D = A₀[1/f]
- The ideal of definition is I·D
- Is 1/f ∈ I·D? Not necessarily. But (1/f)^n = s^{-kn} · (s^k/f)^n,
  and for large k, s^k/f ∈ A₀ ⊆ D. So (1/f)^n ∈ s^{-kn} · D.
  For (1/f)^n → 0: need (1/f)^n ∈ (I·D)^m eventually.
  Since s ∈ I (for large power), s^{km} ∈ I^m, so s^{km} · D ⊆ I^m · D.
  And (1/f)^n = (s^k/f)^n · s^{-kn}. For large n, ...

This analysis shows 1/f IS topologically nilpotent in the localization
topology when f is not a unit of A₀. This is exactly the case for R(1/f)
where v(f) ≥ 1 (f is not "small").

So the universal property DOES give a continuous map A⟨X⟩ → presheafValue D
with T = {1}. The product topology works!

## Revised plan

Given that the product topology (= T-topology for T={1}) already gives:
1. A⟨X⟩ is complete ✓
2. A[X] is dense in A⟨X⟩ ✓ (Prop 5.49(1))
3. Universal property: continuous maps to complete rings when target of X is
   topologically nilpotent (X^n → 0 forces g(X)^n → 0)

The evaluation map A⟨X⟩ → presheafValue D (sending X → 1/f) IS continuous
for the product topology BECAUSE 1/f is topologically nilpotent in the
localization topology. The reviewer's concern was about general power-bounded
elements, but for the specific R(1/f) case, 1/f IS topologically nilpotent.

## Key question to resolve first

Is `invS = 1/f` topologically nilpotent in `presheafValue D` for the
localization R(1/f)?

This needs: `(1/f)^n → 0` in the localization topology on `Localization.Away f`.
The localization topology uses `{image((I·D)^m)}` as neighborhoods. We need
`(1/f)^n ∈ image((I·D)^m)` eventually.

Since `(1/f)^n ∈ D` and `I·D` is an ideal of definition, `(I·D)^m · D ⊆ D`
but `(1/f)^n` might not be in `(I·D)^m` unless 1/f is topologically nilpotent.

For 1/f topologically nilpotent: in the localization A_f, the topology has
fundamental system `{(I·D)^m}`. And 1/f ∈ D. Is 1/f topologically nilpotent?

In the Tate ring A with pair (A₀, sA₀) (s = topologically nilpotent unit):
- D = A₀[1/f]
- I = sA₀, so I·D = sD
- (I·D)^m = s^m · D
- Is (1/f)^n ∈ s^m · D for large n?
- (1/f)^n = (1/f)^n, and s^m · D = {s^m · d : d ∈ D}
- Need: (1/f)^n = s^m · d for some d ∈ D
- Equivalently: s^{-m} · (1/f)^n ∈ D = A₀[1/f]
- = (s^{-m} · f^{-n}) ∈ A₀[1/f]
- = s^{-m} · (1/f)^n

For n ≥ m/k (where s^k ∈ fA₀ or similar), this might work. But it depends
on the relationship between s and f.

Actually, for a GENERAL f ∈ A, 1/f might NOT be topologically nilpotent.
E.g., if f = s (the topologically nilpotent unit itself), then 1/f = s^{-1}
which is NOT topologically nilpotent (it's a unit in A).

So the universal property with T = {1} (requiring target topologically
nilpotent) does NOT work for all f! Only for f such that 1/f is topologically
nilpotent in the localization.

This means we DO need G2-topo with a different T. For R(1/f), use T = {f}
(not {1}). Then the T-topology on A⟨X⟩_{T={f}} has neighborhoods
`{∑ aₙ Xⁿ : aₙ ∈ f^n · U}`. And X is power-bounded (not topologically
nilpotent) in this topology. The evaluation X ↦ 1/f is continuous because
1/f is power-bounded (in D).

So the plan needs REVISION: we need A⟨X⟩_T with T ≠ {1}.

## Revised implementation

For R(1/f): use `A⟨X⟩_{T={f}}` (restricted power series where `aₙ ∈ f^n · U`
for all open subgroups U). This is the same underlying SET as `TateAlgebra A`
but with a DIFFERENT topology.

Define a new type `TateAlgebraT A f` carrying the T-topology for T = {f}.
