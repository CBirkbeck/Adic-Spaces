## Background: Why This Works

In a valuation ring V₀, ideals are totally ordered (`ideal_le_total`). If I has generators s₁,...,sₖ with values g₁,...,gₖ in V₀.ValueGroup (all < 1), then g₀ = min(gᵢ) is the "most nonunit" value, and `(φ(a₀))` (the principal ideal generated by the min-valued generator) contains ALL other φ(sᵢ). Hence:

- `gᵢ = g₀ · rᵢ` with `rᵢ ≤ 1` (in the value group)
- Coarsened: `[gᵢ] ≤ [g₀]` (coarsening preserves ≤)
- So ALL I-generators have coarsened value ≤ `[g₀]` — the bound `g = [g₀]` works for `h_gen`

This is Wedhorn's key insight: by coarsening at the MOST nonunit generator, all other generators automatically fall under the same bound.

Actually, on reflection, the approach using `isContinuous_of_ideal_pow_lt` directly is BETTER than the bound approach, because it doesn't require ALL generators to have value < 1. It only requires the I^n-values to be bounded by [g₀]^n.

The key lemma needed: for a ∈ I^n, the pulled-back value satisfies v₀(a) ≤ g₀^n.

This follows from:
- J = Ideal.map φ P.I = (φ(a₀)) in V₀ (principal, since V₀ is a valuation ring and a₀ has min value)
- J^n = (φ(a₀)^n) = (φ(a₀^n))
- For a ∈ I^n: φ(a) ∈ J^n, so V₀.valuation(φ(a)) ≤ V₀.valuation(φ(a₀)^n) = g₀^n
- v₀(a) = V₀.valuation(φ(a)) ≤ g₀^n
- v'(a) = coarsened(v₀(a)) ≤ coarsened(g₀^n) = [g₀]^n

Actually, this is complex. The simpler approach: since V₀ is a valuation ring and I is finitely generated, J is principal. The principal ideal bound gives the result directly.

- `ValuationRing.dvd_total` (exists in Mathlib: for any a, b in a valuation ring, a | b or b | a)
- Finset.induction to show the ideal generated by a finite set is principal

This is a clean helper lemma.
